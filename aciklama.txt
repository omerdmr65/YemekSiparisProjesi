VTYS Yemek Sipariş Sistemi  Projesi

 Proje Amacı

Bu proje, restoran tabanlı bir yemek sipariş sisteminin veritabanı altyapısını oluşturmak amacıyla geliştirilmiştir. Sistem; müşteri, sipariş, ürün, ödeme ve kurye işlemlerini yönetebilecek şekilde tasarlanmıştır.

1. İlk Aşama

Bu aşamada yemek sipariş sistemi için temel veritabanı yapısı oluşturulmuştur.

2. Oluşturulan Tablolar

1) Musteri
2) Restoran
3) Urun
4) Kategori
5) Siparis
6) SiparisDetay
7) Kurye
8) Odeme

3. Yapılan İşlemler

- Primary Key yapıları oluşturuldu.
- Foreign Key ilişkileri kuruldu.
- Tablolar arası bağlantılar oluşturuldu.
- ER diyagramı hazırlandı.
- GitHub repo yapısı oluşturuldu.

4. ER Diagramı

ER diyagramı docs klasörü içerisinde bulunmaktadır.

5. Kullanılan Teknolojiler

- SQL Server
- SSMS
- GitHub
