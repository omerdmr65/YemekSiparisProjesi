# Yemek Sipariş Sistemi Veritabanı Projesi

Bu proje VTYS-1 dönem projesi kapsamında geliştirilmiş çevrimiçi yemek sipariş sistemi veritabanı tasarımıdır.

Projede müşteri, restoran, kurye, ürün, sipariş, ödeme ve askıda yemek modülleri bulunmaktadır.

Sistem içerisinde:
- Primary Key / Foreign Key ilişkileri
- CHECK / UNIQUE / NOT NULL constraint yapıları
- JOIN, LEFT JOIN, GROUP BY/HAVING ve Subquery sorguları
- View, Trigger ve Index yapıları
- Askıda Yemek sistemi
- 100 adet sipariş verisi

yer almaktadır.

Proje SQL Server kullanılarak geliştirilmiştir.
