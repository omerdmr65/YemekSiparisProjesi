VTYS Yemek Sipariş Sistemi  Projesi

1. Proje Amacı

Bu proje, restoran tabanlı bir yemek sipariş sisteminin veritabanı altyapısını oluşturmak amacıyla geliştirilmiştir. Sistem; müşteri, sipariş, ürün, ödeme ve kurye işlemlerini yönetebilecek şekilde tasarlanmıştır.

2. Kullanılan Tablolar

1) Musteri
Müşteri bilgilerini tutar.

2) Restoran
Restoran bilgilerini tutar.

3) Urun
Restoranlara ait ürün bilgilerini içerir.

4) Kategori
Ürün kategorilerini tutar.

5) Siparis
Müşteri sipariş bilgilerini içerir.

6) SiparisDetay
Sipariş içerisindeki ürün detaylarını tutar.

7) Kurye
Kurye bilgilerini içerir.

8) Odeme
Sipariş ödeme bilgilerini tutar.

3. Veritabanı Özellikleri

- Primary Key ve Foreign Key yapıları kullanılmıştır.
- Tablolar arası ilişkiler kurulmuştur.
- Sipariş sistemi mantığına uygun normalize yapı oluşturulmuştur.
- SQL Server üzerinde geliştirilmiştir.
